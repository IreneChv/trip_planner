# trip_planner

The goal is to create a trip planner. 
From GTFS data, and trip parameters provided by the user, the program returns travel possibilities meeting the constraints. 

Python x SQLite project.

Needs to install:
- python3 + numpy, pandas, sqlalchemy
- sqlite3

Work in progress.
